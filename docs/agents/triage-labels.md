# Triage Labels

Five canonical triage roles used by the `triage` skill:

| Label | Meaning |
|-------|---------|
| `needs-triage` | New issue, not yet categorized |
| `needs-info` | Blocked — waiting on more details from reporter |
| `ready-for-agent` | Clear enough for an agent to pick up and implement |
| `ready-for-human` | Requires human judgment, review, or decision before proceeding |
| `wontfix` | Declined — won't be worked on |
